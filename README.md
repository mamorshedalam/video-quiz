# Video Quiz
### A Complete Single Page Application (SPA) made with Firebase using React JS.  
---

## Installation

1. Clone repository

```bash
https://github.com/mamorshedalam/video-quiz.git
```

2. Go to the folder

```bash
cd video-quiz
```

3. Install Node packages

```bash
npm i
```

4. Run Local server with development change waching...

```bash
npm start
```

5. Open browser

```bash
http://localhost:3000
```

- [x] JavaScript
- [x] React JS
